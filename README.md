# 使用方法如下

![example.gif](https://i.imgur.com/UMXpOQS.gif)

## 打包(已解压)

```shell
bun run build
```
解压后的插件位置: .output\chrome-mv3-build
之后谷歌浏览器开发者模式直接安装插件即可
